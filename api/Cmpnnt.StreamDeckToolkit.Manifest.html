Namespace Cmpnnt.StreamDeckToolkit.Manifest

Classes

ManifestAction

Represents a single action entry within a Stream Deck plugin manifest. Populated automatically by the manifest source generator from [StreamDeckAction] attributes.

ManifestApplicationsToMonitor

Specifies the applications whose launch and termination events the plugin wants to receive. Supply an instance via ApplicationsToMonitor.

ManifestConfigBase

Base class for providing manifest configuration that cannot be derived from code or MSBuild properties. Subclass this in your plugin project to supply default states, encoder configuration, application monitoring, and profile settings. Values from this class take highest priority — they override both [StreamDeckPlugin]/[StreamDeckAction] attributes and MSBuild properties.

ManifestEncoderConfig

Configures the Stream Deck+ encoder (dial) appearance for an action. Set via DefaultEncoder or per-action in the manifest.

ManifestOS

Specifies a supported OS platform and its minimum required version.

ManifestProfile

Describes a Stream Deck profile bundled with the plugin. Supply instances via Profiles.

ManifestRoot

Represents the root structure of a Stream Deck plugin manifest (manifest.json). Populated by the manifest source generator at build time; use ManifestConfigBase and the [StreamDeckPlugin]/[StreamDeckAction] attributes to control the output.

ManifestSoftware

Specifies the minimum required Stream Deck software version.

ManifestStateConfig

Configures the appearance of a single action state in the Stream Deck manifest. A state controls the image, title, and font displayed on the key for that state.

ManifestTriggerDescription

Human-readable descriptions of the dial's trigger interactions shown in the Stream Deck UI. All fields are optional and omitted when null.
