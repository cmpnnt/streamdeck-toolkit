Configures plugin-level manifest properties. Values set here override MSBuild properties but are overridden by ManifestConfigBase. Apply to the assembly ([assembly: StreamDeckPlugin(...)]) or to any class in the project.

[AttributeUsage(AttributeTargets.Assembly|AttributeTargets.Class, Inherited = false, AllowMultiple = false)]
public sealed class StreamDeckPluginAttribute : Attribute
