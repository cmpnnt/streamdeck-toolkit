Class ManifestRoot

Namespace
Cmpnnt.StreamDeckToolkit.Manifest
Assembly
StreamDeckToolkit.dll

Represents the root structure of a Stream Deck plugin manifest (manifest.json). Populated by the manifest source generator at build time; use ManifestConfigBase and the [StreamDeckPlugin]/[StreamDeckAction] attributes to control the output.
